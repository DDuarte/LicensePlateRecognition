RDM by Duarte & Miguel

Reconhecimento Digital de Matrículas

Projecto com o objectivo de criar um software capaz de analisar uma imagem, vídeo ou webcam e identificar as matrículas automóveis presentes. Posteriormente, é feito uma comparação dessa informação com uma base de dados.

Dependencies

Qt OpenCV Tesseract-ocr MySQL
